# FCT

`FCT` is a mathematical operation.

## Description

`FCT` takes 1 integer argument, evaluates its factorial value, and stores the result in the result register.
From there, the result can be assigned to a variable using `ASSIGN`.

## Usage

`FCT <integer>;`

## Examples

`FCT 5;` - Calculates `5!`

`FCT 3;` - Calculates `3!`
